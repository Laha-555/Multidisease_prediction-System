
🩺 Multi-Disease Prediction System using Machine Learning
📌 Project Overview

This project is a Multi-Disease Prediction System built using Machine Learning that predicts the likelihood of three major diseases:

❤️ Heart Disease

🍬 Diabetes

🩺 Kidney Disease

The system uses multiple ML algorithms, proper data preprocessing techniques, and an interactive frontend built with Streamlit.
All model development and experimentation were done using Google Colab.

The goal of this project is to demonstrate an end-to-end machine learning workflow, from raw medical data to a deployed prediction system.

🎯 Objectives

To analyze medical datasets and extract meaningful insights

To apply data preprocessing techniques for better model performance

To build and compare multiple machine learning models

To develop a single platform capable of predicting multiple diseases

To create a user-friendly web interface for real-time predictions

🧠 Machine Learning Algorithms Used

The following algorithms were implemented and evaluated:

Logistic Regression

Used for baseline classification

Simple, interpretable, and effective for binary classification problems

Random Forest Classifier

Used to improve accuracy and handle complex feature interactions

Ensemble-based approach for better generalization

Each model was trained, tested, and evaluated before selecting the final version.

🧹 Data Preprocessing Steps

Data preprocessing played a crucial role in this project. The following steps were performed:

Handling missing values (using forward fill / appropriate strategies)

Removing irrelevant or duplicate data

Feature scaling and normalization

Splitting data into training and testing sets

Selecting important medical features affecting disease prediction

🛠️ Tools & Technologies

Python

Pandas & NumPy – data handling and analysis
